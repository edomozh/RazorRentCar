# To deploy:
1. download and install Windows Hosting Bundle Installer https://dotnet.microsoft.com/en-us/download/dotnet/thank-you/runtime-aspnetcore-6.0.8-windows-hosting-bundle-installer
2. restart IIS
3. start new site

# In the app:
1. SQL Server
2. EF Core
3. Razor pages
4. MVC framework
5. Signal R
6. Serilog
